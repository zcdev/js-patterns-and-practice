# JS Patterns and Practice

This repository contains curated JavaScript examples focused on problem-solving, patterns, and practical decision-making.

The code here comes from hands-on practice and learning exercises, but each example has been refined to reflect how I’d approach similar problems in real projects.

## What you’ll find
- Small, focused JavaScript examples
- Emphasis on readability and intent
- Notes or comments where tradeoffs matter

## What you won’t find
- Full tutorial walkthroughs
- Exhaustive coverage of language features
- Raw exercise dumps

## Related writing
Some examples are accompanied by short learning posts on my blog, where I expand on the reasoning and alternatives behind the solutions.
